Mobile Guide - ໄຟໄໝ້ຄົວ

ไฟล์นี้คือคู่มือสั้น ๆ สำหรับเปิดเว็บบน iPhone และ Android

สำคัญ

ถ้าจะให้ลูกค้าทุกคนเปิดได้ ต้องนำโฟลเดอร์เว็บนี้ไปวางบน Hosting ที่มี HTTPS ก่อน เช่น Netlify, Vercel, Cloudflare Pages หรือ hosting ของร้าน

Drag & Drop ขึ้นออนไลน์

ให้ลากโฟลเดอร์ที่มีไฟล์ index.html, app.js, styles.css, manifest.json, service-worker.js และโฟลเดอร์ images อยู่ข้างในขึ้น Netlify Drop โดยตรง ไม่ต้องลากไฟล์ zip และไม่ควรลากโฟลเดอร์ชั้นนอกที่ไม่มี index.html อยู่ระดับแรก

iPhone / iPad

เปิดลิงก์เว็บใน Safari → กด Share → Add to Home Screen

Android

เปิดลิงก์เว็บใน Chrome → กดเมนู ⋮ → Add to Home screen หรือ Install app

ทดสอบในวง Wi‑Fi เดียวกัน

เปิดเว็บด้วย Live Server ในคอม แล้วใช้มือถือเข้าผ่าน IP ของคอม เช่น http://192.168.1.10:5500
